# 🎰 Baron Bot V15.1 - Global Betting Matrix HQ

**Baron Bot**, otomatik Spor Toto (Football Pool) bahisleri için tasarlanmış profesyonel bir Chrome eklentisidir. Gelişmiş matris indirgeme algoritmalarını gerçek zamanlı otomasyonla birleştirerek kupon maliyetlerinizi düşürürken kazanma ihtimalinizi maksimize eder.

**Baron Bot** is a professional-grade Chrome Extension designed for automated football pool (Toto) betting. It combines advanced matrix reduction algorithms with real-time automation to maximize your winning probability while minimizing total ticket costs.

## 🛠️ Kurulum Rehberi / Installation Guide

* **İndir / Download:** Proje klasörünü bilgisayarınıza indirin ve zipten çıkarın. / Download the project folder and extract the ZIP file.
* **Eklentiler / Extensions:** Google Chrome'da `chrome://extensions/` adresine gidin. / Open Google Chrome and navigate to `chrome://extensions/`.
* **Geliştirici Modu / Developer Mode:** Sağ üst köşedeki **"Geliştirici modu"** anahtarını açın. / Toggle the **"Developer mode"** switch in the top right corner.
* **Paketlenmemiş Yükle / Load Unpacked:** **"Paketlenmemiş öğe yükle"** butonuna tıklayın ve klasörü seçin. / Click **"Load unpacked"** and select the project folder.
* **Sabitle / Pin:** Eklentiyi araç çubuğuna sabitleyerek komuta merkezini aktif edin. / Pin the extension to your toolbar to activate the Command Center.

## 🕹️ Nasıl Kullanılır? / How to Use?

* **Veri Çek / Sync:** Nesine Spor Toto sayfasını açın ve **"Sync Live Bulletin"** butonuna basın. / Open the Nesine (or supported) page and click **"Sync Live Bulletin"**.
* **Strateji Kur / Strategy:** 1-X-2 butonlarını kullanarak 15 maçlık tercihinizi yapın. / Select your 1-X-2 preferences for all 15 matches.
* **Matrisle / Matrix:** **14-Guarantee** modunu seçerek maliyet avantajını görün. / Choose **14-Guarantee** mode for optimal cost reduction.
* **Hesapla / Calculate:** **"Calculate & Lock"** butonuna basarak tüm kolonları üretin. / Click **"Calculate & Lock"** to generate all possible columns.
* **Ateşle / Deploy:** **"DEPLOY TO LIVE"** butonuna basın ve botun otomatik işlemesini izleyin. / Click **"DEPLOY TO LIVE"** and watch the bot handle the placement.

## 🛡️ Güvenlik & Risk / Security & Risk

* **İnsan Taklidi / Human Mimic:** Bot, radara yakalanmamak için paketler arasında ~3.5 saniye bekler. / The bot waits ~3.5s between batches to mimic human behavior.
* **Hata Koruması / Safety:** Operasyon sırasında seçimlerin değiştirilmesini engellemek için sistem kilitlenir. / The system locks during deployment to prevent accidental changes.
